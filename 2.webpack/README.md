## 安装webpack
```
npm install webpack -S
```
## package.json中添加脚本
```
 "scripts": {
    "build": "webpack"
  }
```
## 运行脚本
```
npm run build
```

## 加载CSS
```
npm install style-loader css-loader -S
```

## 加载图片
```
npm install url-loader
```

## 把ES6编译成ES5
```
npm install babel-loader -S
npm install babel-preset-es2015 -S
```
